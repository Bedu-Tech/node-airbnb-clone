# Airbnb Clone
## A simple node application based in Airbnb.com.

### Used tools
- Node.
- Express.js.
- MongoDB & Mongoose.
- Jade for templating.

### Development
```sh
# Install dependencies.
$ yarn install

# Start project.
$ yarn start

# Go to localhost on port 3000 in a browser. Note: If you specified a custom port on environment go to it.
```
